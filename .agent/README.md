# .agent Klasörü

Bu klasör, Google Antigravity platformu ile uyumlu Agent entegrasyonu için ayrılmıştır.

## Temel Bileşenler
- Reasoning model
- Tools
- Artifacts
- Knowledge

## Özelleştirmeler
- Agent Modes / Settings
- MCP
- Rules / Workflows

Daha fazla bilgi için: [Antigravity Agent Dokümantasyonu](https://antigravity.google/docs/agent)

Bu klasöre, Antigravity Agent ile ilgili özel ayar, mod, kural veya entegrasyon dosyalarınızı ekleyebilirsiniz.
